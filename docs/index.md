# NotebookLens Docs

NotebookLens helps DS/ML teams review Jupyter notebook changes on GitHub.

It has two product surfaces:

- the OSS GitHub Action, which adds one sticky PR comment
- the hosted review workspace beta, which opens from a dedicated check run

## Start here

- New to NotebookLens? Start with [Quick Start: OSS Action](quickstart-action.md).
- Evaluating the richer review workflow? Use [Quick Start: Hosted Review Workspace Beta](quickstart-workspace.md).
- Need trust details first? Read [Privacy and Data Flow](privacy.md) and [Troubleshooting](troubleshooting.md).
